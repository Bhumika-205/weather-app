# Weather App

A simple React-based weather application that fetches real-time data using the OpenWeather API.

## Features

* Search weather by city name
* Displays temperature, humidity, and min/max values
* Dynamic weather icons based on conditions
* Basic error handling for invalid input

## Tech Stack

* React (Vite)
* Material UI
* OpenWeather API

## Setup

1. Clone the repository:

```bash
git clone https://github.com/Bhumika-205/weather-app.git
cd weather-app
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file in the root directory:

```env
VITE_API_KEY=your_api_key_here
```

4. Run the app:

```bash
npm run dev
```

## API

OpenWeather API: https://openweathermap.org/api
